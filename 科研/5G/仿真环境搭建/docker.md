## 1 docker

## 2 docker-compose

### 1. 启动和关闭（同时移除容器）

```bash
docker-compose up
docker-compose down
```

### 2. 启动和关闭（不移除容器）

```bash
docker-compose start
docker-compose stop
```

### 3. 列出由 docker compose 管理的容器的状态

```bash
docker-compose ps
```

### 4. 列出日志

```bash
docker-compose logs
```
